## Installation

### Via Moltbot

```bash
# Once packaged and published to ClawdHub
moltbot skills install gandi-skill
```

### Manual Installation

1. Clone this repo or download the `.skill` file
2. Copy `gandi-skill/` to your Moltbot skills directory
3. Restart Moltbot
